📋 Install tag

Dodaj ovaj tag jednom po stranici, idealno u <body>. Placement se konfiguriše kroz Magnite portal (account_id, site_id, zone_id).

HTML — outstream tag (Magnite Apex)
<!-- Lupon Outstream Video --> <script> (function(){ var s=document.createElement('script'); s.src='https://video-outstream.rubiconproject.com/apex.js'; s.onload=function(){ MagniteApex.renderAd({ magnite:{ account_id:YOUR_ACCOUNT_ID, site_id:YOUR_SITE_ID, zone_id:YOUR_ZONE_ID, size_id:203 }, width:640,height:360, placement:{attachTo:'top',position:'after',align:'center'}, collapse:true,closeButton:true }); }; document.head.appendChild(s); })(); </script>
ads.txt — dodaj ovu liniju
rubiconproject.com, YOUR_ACCOUNT_ID, DIRECT, 0bfd66d529a55807 # Magnite Outstream luponmedia.com, YOUR_PUB_ID, RESELLER # Lupon MCM
⚙️ Kako radi
1
Tag se ubaci jednom na stranicu
Magnite Apex JS se učitava async (~12KB). Ne blokira page load.
2
Apex bira placement automatski
attachTo: 'top' + position: 'after' — Apex ubaci video player odmah ispod prvog elementa na stranici. Bez CSS konfiguracije.
3
Viewport trigger — play samo kad je vidljiv
Video se autoplay-a muted samo kad je u viewportu. Scroll out — pauza. IAB viewability standard: 50% piksela, 2 sekunde.
4
collapse: true — no-fill = slot nestaje
Ako Magnite ne vrati bid, player se automatski sakriva. Nema praznog prostora na stranici.
