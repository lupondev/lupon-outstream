⚙️ Kako radi
1
Publisher dodaje jedan div u članak
Div se postavlja unutar tijela teksta — između 2. i 3. paragrafa idealno. Tag je lightweight (~3KB).
2
IntersectionObserver detektuje viewport entry
Kad slot uđe 50% u viewport, player se inicijalizira. Prebid aukcija se pokreće. Nema load-a unaprijed — 0 impact na page speed dok korisnik ne scrolluje do slota.
3
Prebid aukcija → VAST tag → video play
Standardna Prebid.js aukcija sa video mediaTypes. Pobjednički bid vraća VAST URL. Video se autoplay-a muted (GDPR + browser policy compliant).
4
Pause on scroll-out, resume on scroll-in
Korisnik scrolluje dalje — video pauzira. Vrati se — nastavlja. Korisnik može zatvoriti X dugmetom ili ignorisati. Nikad ne blokira sadržaj.
📋 Install tag

Dodaj ovaj kod unutar tijela članka — između 2. i 3. paragrafa. Tag je self-contained, ne treba dodatni JS.

HTML — ubaci u članak
<div class="lupon-inread" data-pub-id="YOUR_PUB_ID" data-site-id="YOUR_SITE_ID" data-floor="0.10"></div> <script src="https://cdn.luponmedia.com/inread/v1/player.js" async></script>
Prebid.js — ad unit config (dodaj u postojeći pbjs setup)
// Dodaj u postojeći pbjs.addAdUnits([]) { code: 'lupon_inread_1', mediaTypes: { video: { context: 'instream', playerSize: [[640, 360]], mimes: ['video/mp4', 'video/webm'], protocols: [2, 3, 5, 6], // VAST 2, 3, DAAST 1, 2 playbackmethod: [2], // autoplay muted placement: 3, // in-article skip: 1, skipafter: 5 } }, bids: [{ bidder: 'rubicon', params: { accountId: 'YOUR_ACCOUNT', siteId: 'YOUR_SITE', zoneId: 'YOUR_ZONE' } },{ bidder: 'criteo', params: { networkId: "YOUR_NETWORK_ID" } }] }
ads.txt — dodaj ovu liniju
luponmedia.com, YOUR_PUB_ID, DIRECT, # InRead video
